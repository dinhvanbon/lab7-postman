01 · Giới thiệu Postman

Postman là nền tảng phổ biến nhất để phát triển và kiểm thử API. Nó cho phép gửi các HTTP request, kiểm tra response, viết test scripts tự động, và tổ chức các request thành Collections để quản lý dễ dàng.

Thành phầnMô tả
Request BuilderNơi cấu hình method, URL, headers, body
Response ViewerHiển thị status code, thời gian, dữ liệu trả về
Test ScriptsViết kiểm thử tự động bằng JavaScript
CollectionsNhóm các request theo chức năng
EnvironmentsQuản lý biến môi trường (URL, token...)
